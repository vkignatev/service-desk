Техническое задание проекта СервисДеск
Слои приложения:
1) БД
2) серверная часть (бизнес логика в сервисах)
3) WEB-морда
++++++++++++++++++++++++++++++++++++++++++++++++

1) Разработка и моделирование/проектирование Базы Данных.
1.1 Продумать сущности/данные
1.2 Продумать связи между ними (данными/сущностями)
1.3 Нормализация, логическая связность бизнеса и данных

2) Функциональность программы
2.1 Регистрация пользователя.
2.2	ЛК пользователя (просмотре своих заявок, создание заявки через ЛК)
2.3 ЛК исполнителя (просмотр назначенных заявок)
2.4 ЛК администратора (меню настройки ПО, все заявки, выжиты)
2.5 Создание заявки через ЛК
2.6 Создание заявки по e-mail
2.7 Бизнес процесс (цикл заявки)
2.8 Отчёты по заявкам
2.9 Уведомление пользователя (ЛК, почта)
2.10 Уведомления исполнителя (ЛК, почта)
2.11 Календарь, празд. дни.

3) UI
